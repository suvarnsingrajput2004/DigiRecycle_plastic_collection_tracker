# ♻️ DigiRecycle – Plastic Collection Tracker

DigiRecycle is a smart plastic waste collection and tracking platform designed to improve recycling management and encourage sustainable waste disposal practices. The system allows users to submit plastic collection requests, track collection activities, and manage recycling operations efficiently.

## 🚀 Features

- User registration and login
- Plastic collection request management
- Track collection status
- Dashboard for monitoring activity
- Waste management workflow
- Responsive user interface
- Sustainable recycling support
